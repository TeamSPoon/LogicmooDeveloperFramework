''[http://cliopatria.swi-prolog.org/ ClioPatria]'' is a Prolog framework
to construct Semantic Web Applications. It provides a programmatic
environment for [[RDF]], [[RDFS]] and [[OWL]], [[SPARQL]], while the
hosting Prolog system supports rule-based inferencing.

ClioPatria builds on top of the ''[http://www.swi-prolog.org/
SWI-Prolog]'' web-server and RDF infrastructure.  It adds right
management, a [[SPARQL]] end-point, [[LOD]] services, web tools
for exploring the loaded [[RDF]] data and a package management
system that allows for sharing code conviently between users.
